CSS Snippet Cheatsheet

Ever have trouble recalling the exact syntax for your favorite CSS code? Give it a permanent home and add it to this page! Select any snippet below and it'll automatically select all of the code for you to copy.

Flexbox flex-box

Use these three propeties to create a Flexbox flex-box layout.

.row { 
    display: flex;
    flex-direction: flex-box;
    flex-wrap: wrap;
}

Flexbox Column

Use this to create a Flexbox column layout.


.column { 
    display: flex;
    flex-direction: column;
}

CSS Grid Layout

Build a 12-column layout using Grid.

.grid { 
    display: grid;
    width: 100%;
    grid-template-columns: 
repeat(12, 1fr);
}

Linear Gradients

This will create a background linear gradient from top to bottom.




.linear-gradient-background { 
    background-image: linear-
gradient (
    rgba(232, 102, 236, 0.3) 0%,
    rgba(232, 102,236, 0.6) 100%
    );
}

Box Transition Glow

Use transition andbox shadows to glow on hover.

.code-card .card-header { 
    border-radius: 8px;
    trasition: all 0.5s ease-in-
out;
}

.code-card:hover,
.code-card:hover .card-header {
    box-shadowL inset 0px 0px 8px 
rgba (232, 102, 236, 1), 0 0 15px
rgba (232, 102, 236, 1);
}
